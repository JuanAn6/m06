JSON

És un format per l’intercanvi de dades. Permet descriure les dades amb una sintaxi dedicada que es fa servir per identificar i gestionar informació. El creador d’aquesta notació és en Douglas Crockford.

JSON va néixer com una alternativa a XML, el seu fàcil ús en JavaScript ha generat un gran nombre de seguidors, perquè és fàcil de llegir i escriure, a més és més, és concís. No obstant, XML és superior tècnicament perquè és un llenguatge de marques, mentre que JSON és simplement un format per intercanviar dades.

Una de les avantatges és que pot ser llegit per qualsevol llenguatge de programació, per tant, pot ser emprat per l’intercanvi d’informació entre diferents tecnologies.

Tant JSON com XML son usats per exactament el mateix, la única diferència es que JSON és la versió "lleugera" de XML. Consumeix menys recursos “parsejar” (analitzar sintàcticament) un JSON que un XML.

Encara que sembli que JSON es fa anar per JavaScript, JSON és un format d’intercanvi de dades independent del llenguatge de programació.

Els arxius amb informació en aquest format acostumen a tenir extensió .json.

Usos de JSON

  • Per escriure aplicacions JavaScript
  • Per transmetre dades entre el servidor i aplicacions web
  • Serveis Web i API utilitzen el format JSON per a proporcionar les dades públiques
  • En llenguatges de programació moderns (PHP, PERL, PYTHON, RUBY, JAVA).

Tipus de dades JSON

Dins de les dades JSON, es poden distingir dues estructures que es poden enllaçar entre elles sempre que calgui:
  • Objecte JSON:Una col•lecció de parelles nom:valor. És el què seria un registre, tupla o estructura de C o un objecte en C++. Exemple: 
    
                // sintaxi per definir camps nous
                 { key : value }
                // exemple
                  let data = 
                            {
                                nom : "Ramon",
                                cognom_1 : "Guerra",
                                unio: 2012
                            };
                // accés
                  console.info(data.unio);
  • Array JSON: Una llista ordenada de valors (Un array). Veure exemples en els arxius adjunts: json1.json i script.js

Sintaxi de la notació JSON

Un objecte Json és un conjunt desordenat de parells nom/valor. Un objecte comença amb { clau d’obertura i acaba amb } clau de tancament.
Cada nom es segueix per : dos punts i els valors es separen per , coma.

Si es vol definir arrays en JSON ho faríem començant pel [ claudàtor esquerra i l’acabaríem amb ] claudàtor dret. Els valors de l’array separats per , coma

JSON a la pràctica

En la pràctica JSON s’utilitza per a crear estructures de dades en bases de dades orientades a fitxers (no a taules).
La majoria de bases de dades de les xarxes socials estan basades en fitxers perquè són més ràpides al no haver de duu el control de les integritats referencials.

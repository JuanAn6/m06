Recorregut pels camps d'un objecte JSON

Suposem el següent objecte:

        const items = {
              'first': new Date(),
              'second': 2,
              'third': 'test'
            }

Object.keys(objecte)

Retorna un array amb els noms dels camps de l'objecte JSON. 

            Object.keys(items) = ["first", "second", "third"];
        

        
Object.entries(objecte)

        
Retorna un array, on cada posició és un array de 2 posicions:

        En la primera posició hi ha el nom del camp i en la segona el valor.
        

        
            Object.entries(items) = [["first", Thu Nov 21 2019 18:30:26 GMT+0100 (Hora estàndard del Centre d’Europa)],
                                     ["second", 2],
                                     ["third", "test"]
                                    ]
        

        

		
Més exemples

        
Object.values(objecte)

        
Retorna els valors de l'objecte JSON en un array
        

        
            Object.values(items) = [Thu Nov 21 2019 18:30:26 GMT+0100 (Hora estàndard del Centre d’Europa),
                                    2,
                                    "test"
                                    ]

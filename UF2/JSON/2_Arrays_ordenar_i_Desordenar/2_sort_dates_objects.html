Ordenar Array de JSON's x camp de tipus Date

Suposar l'array "activities" amb el camp "date" que conté objectes de tipus Date que es volen ordenar. La primera crida a sort és vàlida, però tant "activities" com "sortedActivities" queden ordenades pel camp "date". Amb la segona opció, fent ús del mètode slice s'aconsegueix que l'array original no acabi també ordenat.

        
            const activities = [
              { title: 'Hiking', date: new Date(anyActual+'-06-28') },
              { title: 'Shopping', date: new Date(anyActual+'-06-10') },
              { title: 'Trekking', date: new Date(anyActual+'-06-22') }
            ]
            
            const sortedActivities = activities.sort((a, b) => b.date - a.date)
            
            const sortedActivities = activities.slice().sort((a, b) => b.date - a.date)

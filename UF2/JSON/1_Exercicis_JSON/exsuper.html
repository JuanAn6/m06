S’adjunta l’arxiu superheroes.json, dins seu hi ha una variable JSON que cal tractar.

En carregar aquesta web, mostrar dins l'etiqueta out les dades de la variable JSON segons s'indica tot seguit

Primer una taula:

  1. La taula tindrà 2 columnes i 5 files (no cal titol per les columnes)
  2. La primera columna amb els noms dels 5 primers camps de la variable JSON.
  3. La segona columna amb els valors dels 5 primers camps de la variable JSON.

Segon: mostrar la informació de l'array members, en concret, per cada posició de members mostrar els 3 primer camps, dins d'una estructura amb etiqueta dl, cada nom de camp en una etiqueta dt i cada valor en una etiqueta dd Veure exemple d'ús de l'etiqueta dl i les seves filles: dt i dd).

Tercer: mostrar el camp "powers" de cada members (s'ha de mostrar després de cada dd de cadascun dels members) en forma de llista ordenada, etiqueta ol, amb els noms dels "powers" ordenats alfabèticament.

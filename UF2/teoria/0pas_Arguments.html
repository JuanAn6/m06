UF 2 - FUNCIONS

Documentació MDN sobre funcions
Habitualment tenen un nom (encara que en JavaScript això és opcional) i es poden cridar (si tenen nom) en qualsevol lloc del codi JS.
Les funcions poden o no retornar valors a l’element que l’ha cridat, amb la paraula clau return.
Els valors retornats, com es veurà més endavant, poden ser funcions o variables que continguin funcions.

DECLARACIÓ DE FUNCIONS

La manera més normal de declarar una funció: 

                
                    function funcio1()
                    {
                        // instruccions a executar
                        return valor_a_retornar; // si cal retornar quelcom
                    }

Es poden passar arguments entre parèntesis, però també hi ha altres maneres que es veuran tot seguit. 

                
                    function funcio2( argument1, argument2, ...)
                    {
                        // instruccions a executar
                        // ús dels arguments que es passen
                        return valor_a_retornar; // si cal retornar quelcom
                    }

NOM DE LES FUNCIONS

És adequat que les funcions tinguin de nom una estructura semblant a les següents propostes: 

                    f_nom
                    fNom
                    func_nom
                    funNom
per tal d’evitar confusions amb noms d’altres elements de la web: noms de variables, noms d’atributs d’etiquetes,....

PAS DE PARÀMETRES

Tipus de pas de paràmetres en funcions:

  • Per valor: implica que si es passa una variable, els seus canvis només seran coneguts dins la funció. Al retornar al bloc de codi des d'on s'ha executat la funció, les variables continuaran amb els valors que ja tenien abans.
  • Per variable/o per referència: implica que si es modifica la variable dins la funció, els canvis són coneguts fora d’aquesta.

Classificació dels tipus de dades en JavaScript:

  1. Variables complexes / Objectes: Array, Date, ... i qualsevol objecte definit per l'usuari.
  2. Tipus de dades primitius: Boolean, null, Undefined, Number, String, Symbol.

Els tipus de dades complexes es passen per referència i els primitius per valor.
Els paràmetres primitius (com pot ser un número) es passen a les funcions per valor; el valor es passa a la funció, però si la funció canvia el valor del paràmetre, aquest canvi no es reflexa globalment o en una altra crida a la funció.
Si es passa un objecte (p. ex. un valor no primitiu, com un Array o un objecte definit per l’usuari) com a paràmetre, i la funció canvia les propietats de l’objecte, aquest canvi és visible des de fora de la funció.

RECURSIVITAT

Igual que altres llenguatges de programació, JS permet fer crides recursives a funcions. 

                
                    function f_recursiva(x)
                    {
                        if (x == 1) return x;
                        else
                            return x*f_recursiva(x-1);
                    }

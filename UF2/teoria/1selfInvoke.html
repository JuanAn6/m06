UF 2 - FUNCIONS

CRIDA / INVOCAR UNA FUNCIÓ

Per cridar una funció es posa el seu nom en qualsevol part del programa. Si la funció reclama arguments, s’hauran d’omplir amb valors: constants o variables.
Exemples:

                
                    funcio1();
                    funcio2(arg1,arg2,"valor literal",2);
Si les funcions retornen valors, aquests es poden recollir amb l'ús d'una variable:
                
                    let resultat = funcio1(arguments);

Si cal cridar més d’una funció al mateix temps, des del codi HTML es pot fer així: 

                
                    <body onload='lletres(),canviar_epigrama()'>
Però cal evitar incloure en el codi HTML codi JS. Per tant: 
                
                    document.addEventListener('DOMContentloaded',function () {
                        lletres(); canviar_epigrama();
                    });

TIPUS DE FUNCIONS: 1. FUNCIONS AUTOEXECUTABLES (SELF-INVOKING)

Aquestes funcions s'executen una vegada són llegides i no es poden invocar en cap més lloc, siguin o no anònimes: 

                
                    (function funcioAuto() {
                        // instruccions;
                    })();
alternativament, amb notació arrow: 
                
                    ( () => {
                        // instruccions;
                    })();
alternativament, amb notació ;! enlloc de () 
                
                    ;!function () {
                        // instruccions;
                    } ();
Els parèntesis del final indiquen a JavaScript que ha d’executar la funció un cop llegida. I també serveixen per passar paràmetres.
OBSERVACIÓ: Aquest tipus de funció s'executa en el moment que es llegeix l'arxiu .js on estan ubicades. Per tant, pot ser abans o després que el codi HTML hagi estat carregat, tot depenen d'on s'inclogui el codi js.

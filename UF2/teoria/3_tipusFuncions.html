TIPUS DE FUNCIONS

S'han vist les funcions autoexecutables, però també existeixen altres tipus de funcions.

FUNCTION EXPRESSION i FUNCTION ARROW

Les funcions com expressions són les que es fan servir en la notació arrow. Es poden definir amb o sense nom. 

                
                 const add = function (a,b)
                            {
                                return a+b;
                            }
Altrernativament, amb notació arrow: 
                
                 const afegir = (a,b) =>  a+b;
I si tenen més d'una instrucció, amb notació arrow: 
                
                 const potencia = (a,b) =>
                    {  
                        let res = 1;
                        for (let i=0;i<b;i++)
                        {
                            res = res*a;
                        }
                        return res;
                    }

FUNCIONS CONSTRUIDES

També es poden crear funcions fent ús del constructor new. Però el seu ús no és molt recomanable. Un constructor és un element de la POO que ens permet reservar la memòria dinàmicament de l’element que volem crear. 

                
                 const add = new Function('a','b','return a+b');
                 const frasePersonal = new Function('a','b','return alert("què li passa a la "+a+" amb la "+b);');
A dins del constructor, es poden posar els arguments que es vulguin. JavaScript sempre interpretarà que l’últim argument són les línies de codi de la funció.

FUNCIONS COM A PARÀMETRES

De fet ja s'ha fet ús de les funcions com a paràmetre en els temporitzadors setInterval i setTimeout. Exemple: 

                
                 function map(f,a) {
                      let result = [];
                      let i;
                      for (i = 0; i < a.length; i++)      
                        {
                        result[i] = f(a[i]);
                        }
                      return result;
                    }
                
                const multiplicar= function(x) { return x * x * x;}
Exemple de crida a la funció que té una funció com a paràmetre: 
                
                map(multiplicar, [0, 1, 2, 5, 10]);
retorna 
                
                [0, 1, 8, 125, 1000]

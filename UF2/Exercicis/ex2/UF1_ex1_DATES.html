EXERCICI 1. ARRAY DE DATES

PART A

Dissenyar una funció que donada una data i un valor enter positiu, creï un array amb les dates compreses entre la data donada fins a la data més el valor enter positiu indicat, incloent-hi la data donada. Retornar aquest array. Cal suposar que els paràmetres s’han validat anteriorment. Fer ús de la variable arguments per fer ús dels paràmetres.

PART B

Quan es carregui la pàgina web invocar la funció dissenyada en la Part A passant els paràmetres d’un dels jocs de prova i mostrar l’array dins l’etiqueta HMTL <p> amb id "out", però es vol mostrar una data cada segon, i la data s’ha de mostrar invocant el mètode .toLocaleDateString() per tal que es visualitzi com en els exemples dels jocs de prova.

PART C

Quan es carregui la pàgina web i fent ús del mètode forEach recòrrer l'array proporcionat JDPEX1 i invocar la funció dissenyada en la part A passant els paràmetres de les posicions 0 i 1 de cada element de l'array. Mostrar la sortida de la funció per consola. Mostrar les dates invocant el mètode .toLocaleDateString() per tal que es visualitzi com en els exemples dels jocs de prova.
